    Metric Guide
    
                  
    What You Are Looking At
    
                
    
              
    
              
    
                
    
                  
    Server Accuracy
    
                  
    This is the main task performance of the global model on evaluation data. If accuracy stays high under attack, the defense is likely absorbing the disruption. If it drops sharply, the server may be accepting harmful updates.
    
                
    
                
    
                  
    Server Loss
    
                  
    Loss shows how confidently wrong or right the global model is. It can spike before accuracy collapses, so it is useful as an early instability signal even when the top-line accuracy still looks acceptable.
    
                
    
                
    
                  
    Backdoor ASR
    
                  
    Attack Success Rate tracks whether poisoned behavior is actually being implanted. A high ASR means the model is being manipulated even if normal accuracy still looks strong, which is why this matters separately from accuracy.
    
                
    
                
    
                  
    Client Train Loss
    
                  
    This aggregates how local client training is behaving during the round. Rising or unstable train loss can indicate optimization stress, drift between clients, or the effects of active poisoning pressure.
    
                
    
                
    
                  
    Poisoned Examples
    
                  
    This shows how many training examples were actually poisoned in the participating clients. It helps connect the configured attack to what was injected in practice instead of only relying on outcome metrics.
    
                
    
                
    
                  
    Why Train Loss and Poisoned Examples Share a View
    
                  
    They are paired so you can compare cause and effect. Poisoned examples reflect attack pressure, while train loss shows whether that pressure is visibly changing client-side optimization behavior in the same time window.
